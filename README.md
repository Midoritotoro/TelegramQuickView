# TelegramQuickView


<details>
  <summary>Предупреждение!</summary>
	Приложение работает только на Windows 10 версии и выше.
</details>


## Запуск

<details>
  <summary>cmd</summary>

  ```
  git clone https://github.com/Midoritotoro/TelegramQuickView
  cd TelegramQuickView
  mkdir out
  cd out
  mkdir build
  cd build
  cmake ../../
  cmake --build .
  start TelegramQuickView.exe
  ```
</details>
